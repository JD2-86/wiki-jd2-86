= Lesson 2: Apache Maven. Intro

== Practice

* Установить *Apache Maven* (обязательно в директорию: `/opt/apache-maven/${version}`).

=== Task 1

* Сгенерировать проект с помощью команды: `mvn archetype:generate`.
* Найти и добавить в зависимости *JUnit 5*.
* Добавить тесты в проект.
* Поочередно, с помощью CLI, выполнить ниже приведенные команды и убедиться что они выполнили возложенную на их функцию:
** `mvn compile`
** `mvn clean`
** `mvn test`
** `mvn clean`
** `mvn package`
** `mvn clean`
