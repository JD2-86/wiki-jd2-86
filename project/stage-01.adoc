= Stage 00: Инициализация проекта

На подготовительном этапе прорабатывается идея проекта и его особенности. После чего необходимо инициализировать git-repository для проекта.

== Tasks

* Создать remote repository для проекта в *Organisation*: link:https://github.com/orgs/JD2-86/[JD2-86^]
* Добавить в проект файл `README.adoc`, который содержит описание проекта. Пример шаблона приведен ниже.

.Шаблон `README.adoc` для проекта
----
= ${your-project-name}

== Description

==== Какою проблему данный проект решает или что нового приносит для пользователей?

* Пункт 1
* Пункт 2
* ...
* Пункт N

==== Потенциальные пользователи

* Пункт 1
* Пункт 2
* ...
* Пункт N

==== Аналогичные приложения

* Пункт 1
* Пункт 2
* ...
* Пункт N

==== Конкурентное преимущество

* Пункт 1
* Пункт 2
* ...
* Пункт N
----
