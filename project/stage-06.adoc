= Stage 06: Разработка Presentation Layer

На этом этапе создает *Presentation Layer* (*слой представления*) для приложения, который будет каким-то образом отображать данные для клиента.

== Tasks

* Реализовать *navbar* общий для всех страниц, для удобной навигации в web-приложении.
* Реализовать все необходимые страницы в рамках текущей версии проекта.
* Улучшить внешний вид web-приложения за счет использования toolkit *Bootstrap*.
