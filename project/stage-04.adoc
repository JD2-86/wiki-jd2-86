= Stage 04: Инициализация maven-проекта

На этом этапе создает Apache Maven проекта в рамках которого будет выполняться разработка. В процессе разработки может происходить:

* Декомпозиция проекта на модули
* Декомпозиция проекта на независимые модули (сервисы)
* Изменение используемых libraries, tools, stack

NOTE: Для независимых сервисов, чаще всего, практикуют подход *One repository per One service*. В данном случае он не практикуется, так как это усложнит разработку.

== Tasks

* Инициализировать web-application с помощью Apache Maven.
* Создать общедоступную страницу с проектом, которая содержит приветствие для потенциальных клиентов.
